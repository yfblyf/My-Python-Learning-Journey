### 🎂 破壳日计算器 (`1.1py`)

**功能描述**  
通过用户输入的出生年月日信息，计算该日期在一年中的第几天。

**实现思路**
1. 使用 `input()` 函数获取用户出生的年、月、日信息
2. 创建月份天数列表，预设每个月的初始天数
3. 判断是否为闰年，动态调整二月份的天数
4. 运用循环累加计算出生月份之前的总天数
5. 加上出生日的天数，输出最终结果

**核心代码逻辑**
```python
# 月份天数列表
months_days = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

# 闰年判断
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    months_days[1] = 29  # 闰年2月29天

# 计算总天数：提前将a赋值为天数，则只需将出生月份以前的天数总体相加，相加的程序由循环遍历实现
a=day
for i in range(month-1):
    a+=count[i]
```
